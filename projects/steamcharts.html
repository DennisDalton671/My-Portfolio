About SteamCharts

SteamCharts was inspired by web-based tracking sites and designed as a mobile-first alternative to check real-time player statistics for Steam games. This was one of my first Android apps built using Android Studio & Kotlin, where I worked with APIs and databases for the first time.

  • 📊 Live Player Tracking – Displays real-time Steam player counts.
  • 🔍 Game Insights – Provides game descriptions, reviews, and pricing.
  • 💰 Discount Alerts – Shows game discounts and special offers.

Challenges & Lessons Learned

One of the biggest challenges I faced was parsing Steam's API data. Their JSON responses often nest data inside multiple layers, which made accessing useful information (player count, description, price) frustrating.

  • 📌 Parsing Nested JSON – Steam stores its data inside another JSON object, making extraction tricky.
  • 🗃 Working with Databases – First time implementing data storage & retrieval in an Android app.
  • ⚡ Optimized API Calls – Learned how to minimize API requests to avoid unnecessary data fetching.
