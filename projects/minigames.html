The MiniGames app is a personal passion project designed as an ongoing way to learn and experiment with different programming techniques. The goal was to create a structure where I could easily add new games whenever I felt like learning something new or just having fun. Each mini-game serves as a learning experience, helping me refine various aspects of game development, including UI design, physics, and logic systems.
